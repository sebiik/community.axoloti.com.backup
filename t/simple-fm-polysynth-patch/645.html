When I make something more complex, for example a synth with everything(oscs,, envelopes filters, lfos, effects, etc.. I make a subpatch for each of the parts of the synth. For example one for the oscillators, one for the Lfo, one for the filters, etc. Or if it is really complex make one for each oscillator, one for each filter, effect, etc) This way you can place them in a little bit more meaningfull fashion that when just using one subpatch for the whole instrument.
