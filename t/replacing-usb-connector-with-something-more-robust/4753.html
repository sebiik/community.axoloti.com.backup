Ersatzplanet 2018-07-06 15:17:05 UTC #2

I don't see any problems with using a different connector with the same number of connections as the Micro-USB connector. You just have to match pin-for-pin and hand wire the connector since the spacing will never match. The problems with some connectors is that they are specified for different current delivery. There may be a problem with the type-c connectors which are designed to do the job of both a USB-A and USB-B connector so using one on the Axo would be replacing both of the USB connectors.
