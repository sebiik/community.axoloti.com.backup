To be honest I was a bit puzzled why you were using an LFO to trigger the envelope. I thought there was maybe some very specific reason you were doing that. lokki 2018-03-21 07:37:16 UTC #12

this is an old topic, but i just did this:

Lokki contributions

lokki/env/scahd slow (cycling) ahd module. ok, this is an ahd module that is REALLY slow at maximum settings and still quite slow at minimum settings use it for noise sweeps or similar. controls: a = attack time d = decay time divd = divisor of decay time, so that you can get faster decay times if needed cycle = enable/disable cyclemode. h = holdtime, in cyclemode this control sets the hold time at maximum before descending again. has no effect if cyclemode is disabled …

basically a VEEERY SLOW cyclable ahd with variable hold time in cyclemode (ramp up, stay there, ramp down...)
