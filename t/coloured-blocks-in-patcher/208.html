toneburst 2016-03-07 14:56:19 UTC #3

Quartz Composer has coloured areas called Notes that sit underneath the objects. The top of the Note can have text added to it and can be dragged to move the note around. The bottom-right corner can be dragged to change the Note size. You can right-click to change the note colour.

It's a simple but very useful way to visually group collections of objects.
