thetechnobear 2017-03-25 10:25:07 UTC #2

Power each one up in turn.
You can then give it a name in the select device dialog, then this is displayed when you connect to it in the future.
Note; this friendly name is stored on the axo prefs so needs to be done on each computer you use it one.

On MacOS I also rename the midi device to match

( and put a small label physically on each axo case )
