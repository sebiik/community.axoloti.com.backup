thetechnobear 2015-08-12 22:45:48 UTC #2

sounds nice...

some thoughts,
- why are you using midi/ctrl/mpe
rather than midi/in/keyb? is this later for polyphonic expression?
(in which case you may still be better off using keyb/pb etc as you can still use polyphonic multichannel mode.
(mpe is mainly useful for being able to MPE controllers which can send in pitchbend ranges via NRPN)
- mix 1? you could probably use a simple math/*c (which you could subst later with a math/* if you wanted an inlet)

But the thing I did to the patch to have some fun with it , was make it polyphonic
just cut all the objects, except you volume/chorus/audio
add a patcher object, ,
edit
paste
add an audio outlet.
connect the vca to the outlet
edit patch settings, set to polyphonic
close patcher
hit update
attach audio output to your volume

bang... poly bliss

(but change your mpe object to a keyb first, otherwise you may have an issue )
