I do understand the second question ( I think ) and actually had posed it myself before...

basically it is , can you reboot the axoloti, without having to unplug it. (the answer is currently no)
there are a couple of scenarios where this is useful, and I suspect the one the OP is talking about, is if you upload a patch to the SC card and you want to run it, and be disconnected from the UI. (or run the startup patch)

I also don't always like this LIVE mode, and the concept that it is directly related to the patch running or not,
id prefer it was a bit more evident that the UI was 'optionally' attached to the UI, which if you play a little bit you will actually find is true. (e.g. if you stop the axoloti UI without turning LIVE off, you will find the patch is still running, which is exactly how I want it!)

BUT... I will say, I can understand its appeal from a UX point of view in particular for users, used to VSTs and alike, here the live mode feels quite natural, and the way the patch on screen relates directly to what your hearing
