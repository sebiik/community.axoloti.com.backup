I added explicit support for the Virus TI ... I enjoy using its 'Remote' mode, which allows you to assign CCs to all the knobs. unfortunately not the buttons though.

I dont know if you have used this before? if not, basically configure it using the VST which connected to a computer, then the Virus will save it, so when you connect the Virus directly to Axoloti, your good to go
(theres some info in the manual, or virus forum iirc.)

and as MrBim says an iPad with a camera connection kit will also work.
quite a few tools out there, I'm sure some free ones... my favourite, but very deep is Lemur.

the only thing, without coding, your going to be limited on is 'user feedback', as you can don't have a LCD screen to give info back, this is fine for faders/knobs etc, but if you start wanting to do things like change patches...

something fun, some of the launchpad type devices are also small budget... and might be able to provides some feedback.
