gao/dist/biscuitage_m.axo

Status
Ready to use

Description

A simple bitcrucher, with :

  • resolution reduction to 8 bits ;
  • possibility to mute each of the 8 bits ;
  • auto DC offset reduction ;
  • parametric noise gate.

With help patch.
