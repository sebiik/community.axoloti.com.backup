DrJustice 2016-11-16 21:32:31 UTC #5

Note that drj/audio/out_stereo_vol offers 12dB gain. So with the volume knob above 12 o'clock, the output is actually amplified. That is when the readout to the left of the knob id above "x0.250" pr above "-12dB".

There is also a stereo volume control, drj/audio/stereo_vol, in two versions. One with an input to control the volume, and one with a knob. These have no gain.
