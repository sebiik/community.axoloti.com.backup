This would be useful for both my K-Mix sending through DIN and Novation Circuit sending through USB or DIN.

To clarify:
The Novation Circuit has fixed MIDI Channels - Macro Knobs send on channels 1,2,10 etc...
K-Mix allows you to select MIDI channels but it needs an exclusive channel for each of the three MIDI pages.

What this means:
Given that each Axoloti patch receives CC on a single MIDI channel only it limits the amount of MIDI control available per patch.

What would be better:
Being able to select MIDI channel 1-16 and Omni in the CC Modulation window for the controls on each module.
