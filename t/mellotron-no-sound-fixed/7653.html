hitachii 2022-03-12 08:34:19 UTC #2

I don't have the Axocontrol but check the axoloti monitor to see if the files aren't loading. if you press a key and it says "cannot load file: [mellotron file] you might have to move the files to the main SD card folder. it depends on what folder the table/load object is set to.

if the axocontrol mellotron patch is similar to the mellotron patch in the library (or if it's the same one, this is from memory) then go into the mellotron subpatch, that's where you can check the table/ load object to see.

if you're not getting audio it makes sense why the oled is just a flat line. if you press a key and you don't get an error, hook up a disp/scope to any/all audio outputs to see if sound happens and where it stops.
