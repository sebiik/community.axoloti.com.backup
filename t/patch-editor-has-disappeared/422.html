johannes 2015-09-06 23:05:01 UTC #2

First time I'm hearing this.
The last window position and size is saved in the patch file, and when loading this location is restored.
As a quick'n'dirty workaround, you could open a patch file in a text editor and edit the XML, it's at the end of the file

<windowPos>
  <x>291</x>
  <y>118</y>
  <width>1030</width>
  <height>719</height>
</windowPos>

Are you using a multi-screen setup?
