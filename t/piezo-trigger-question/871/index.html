matthewcieplak 2016-01-13 08:25:35 UTC #2

I played around with getting piezo inputs to to work with my e-drum kit (and ended up going with an arduino instead). If you want to use the regular analog in pins, you'll probably want a simple input filter circuit using an op-amp to preamp and limit the peaks of the signal. @timvets mentioned he'd used some piezo disks via the regular analog pins in this thread: http://community.axoloti.com/t/analog-inputs-for-trigger-to-midi-capability/604
