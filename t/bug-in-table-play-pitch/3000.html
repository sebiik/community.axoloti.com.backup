johannes 2017-03-03 15:41:12 UTC #2

I can see that, intuitively, this parameter goes the wrong direction.
But there is a reason for this, the pitch parameter is to adjust the reference pitch of the contents of the table, while the pitch inlet is to control the playback pitch. So if the content of the table is note A5, the pitch inlet left unconnected (default to E4), and the pitch parameter set to A5 - matching the content of the table, it will be transposed downwards, tuned to E4.
Does that makes sense now?
