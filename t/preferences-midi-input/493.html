johannes 2015-09-23 12:55:27 UTC #2

Routing MIDI through the Axoloti editor was implemented before the Axoloti firmware exposed a (class compliant) MIDI device port. You can send MIDI from other application directly to this port, routing MIDI through the editor is no longer needed. This preference/setting is already removed in git.
