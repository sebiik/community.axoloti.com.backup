sevo 2018-06-03 20:44:58 UTC #3

IIRC you can assign individual channels and control numbers to each controller on the nanokey Studio, so you would not have to split messages, but only find a way to feed the same MIDI signal to both devices. Where any MIDI Thru Box would do (e.g. a https://midisizer.com/midithru/midithru-original/ if you are into DIY). If there is not a lot going on (beware, if you use the Chaospad, there probably is) you could even chain the Axolotis, modifying the patch on the first Axoloti in the chain to feed MIDI in to MIDI out.
