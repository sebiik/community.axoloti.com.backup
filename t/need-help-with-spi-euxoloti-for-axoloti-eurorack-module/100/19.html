euxoloti v1 back.png1440x915 603 KB

Some additional infos:
Any I/Os (pots, jacks etc.) will be on a separat PCB.

8x CV-INs (max. +/- 5V voltage input) (JP-CV)
8x Pots (JP POT)
2x Audio input (10ppV to line level) (JP5)
2x Audio output (line level to 10ppV) (JP5)
1x extension headers (JP4) to access Axoloti's pinouts or mount the Axoloti core to a separate frontplate.

optional
gate in/out
LEDs
