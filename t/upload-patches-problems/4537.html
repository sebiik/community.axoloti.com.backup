jaffasplaffa 2018-05-07 16:30:22 UTC #2

@lotwenzel

If you just want to upload a single patch to axoloti you can do that by:
1. Open the patch.
2. Go do drop down menu "patch".
3. Select "upload ot sd-card as startup".
4. You now have a single stand alone Axoloti instrument

If you want to upload more patches, you could try the "patch bank" system, which you can find info about by searching. Or start from this thread:

Patch Changes and File structure in sd card

I have added an object "patch/bankindex" to obtain the index of the last loaded patch via "patch/load i" and a few ready-made "controller objects", for version 1.0.10, you get them via "sync libraries". I cant help you with the specifics on using a patch bank, since I havent used that myself.
But the single patch is really simple to do.
