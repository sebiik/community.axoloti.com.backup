I think that this could be a game changer for creating a proper 3+ OSC subtractive synthesizer emulation, like i.e. the ubiquitous Minimoog (sorry for this example ^_^' )
Choose between the waveforms of each OSC generator could be awesome.
Currently if you wish a selection between a saw, ramp +/-, sine, square and noise, you have to charge the CPU with 6 running oscillators + a selection mixer.
With a switchable waveform OSC you can spare 5/6 of CPU power each generator!

For this project, for example http://www.synth-project.de/axcore.html it is almost impossible to obtain a decent polyphony with one axoloti only.

Also a swithcable custom wavetable OSC would be interesting.
