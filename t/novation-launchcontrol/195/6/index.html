thetechnobear 2015-08-12 10:03:06 UTC #5

@DigitalStains, just to confirm...
Connecting the Axoloti board to a computer is completely independent from using a midi controller attached to the host port. if you wish to use Axoloti 'standalone' you will need to upload the patch to SDRAM or SDCard (and run on startup)

(I mention this, as you PID/VID warning, is from not having Axoloti powered up to the UI, and is just saying the UI cannot talk to the board, handly surprising if its not turned on )
