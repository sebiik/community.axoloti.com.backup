Hugo 2017-02-15 14:41:17 UTC #6

I needed this for my synth in the past and had written an object. I publish it on this occasion:

Hugo Contributions

hug/math/octaveShift.axo A simple math module, that is useful for octave shifting. The two trigger inputs can be used to scale up and down. The result will have a corresponding output like ... -12. 0, +12. +24... Simply add this to the midi note signal that you want to shift. The trigger output is active when both input triggers are active. So you can trigger something else by pressing both buttons at the same time.
