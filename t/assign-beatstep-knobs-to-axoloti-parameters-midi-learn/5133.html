dontjudgedave 2018-11-13 19:48:34 UTC #2

You'll need to know which CC number each knob is on the Beatstep then when you right click on a parameter for in the Axo software you can go to the MidiCC option and you'll select the number there that matches the number of the knob you want to use.

I am not sure the default CC numbers Arturia assigns the knobs/pads but If it is similar to programming my Akai MPK Mini, they should have supplied software that will allow you to assign all of this so you can then use them with the axo (as well as set if the pads are momentary or latching)

so..

  1. Find out the CC numbers already assigned to your knobs/pads - do this either with supplied software or the DAW will probably note which number it is
  2. Note the number of what you want to use on the axo and then right click the parameter for what you want to control and set the midiCC to that number
  3. Set the patch Live and you should be good to go!
jaffasplaffa 2018-11-13 21:29:14 UTC #3

The easiest way is probably:

  1. Connect the beat step to Axoloti.

  2. Then load one of these objects: rbrt/midi/in/monitor

  3. Try moving some of the knobs in the beatstep and see which numbers the midi monitor object shows.

  4. If knob1 for example is cc1 on beatstep, then in Axoloti, you right click on the parameter you want to assign to that knob and select cc1.

  5. Repeat step 3&4 until done.
