I’ve decided to start a topic / challenge for the creation of a collaborative “General Midi” library.

The aim is to have a bunch of patches, easy to use, and useful to learn understand basic axoloti patching. It would be great to cover all the main sounds included in the GM 1 sound set, but it’s only names, so we’re very free to do whatever it inspires us. It’s not very important to be realistic, everybody has probably very different visions of what a synthesized acoustic patch may sound like, many sounds are synth oriented too.
All kind of synthesis are welcome, and we can propose multiple patches for the same patch number (we can never have enough warm pads).
If you use wavetables or samples, you need to host them somewhere, and link here, with your patch.
You can share your patch by using your contrib patch folder or host it directly on this thread (or both).

Lets try and keep the patches simple to use (midi note, velocity, note on, mod wheel #1, pitch bend, aftertouch... if needed). They should respect a certain logic concerning polyphony (a piano’ish sound should be playable with at least 8 voices, but a bass a lot less)

I will update this post with links to the thread post that includes your patch.
