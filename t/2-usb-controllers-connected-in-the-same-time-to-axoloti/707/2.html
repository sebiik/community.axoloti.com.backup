Any way of using two controllers?

The micro-USB port is device only. And the host port does not support USB HUBs. So currently there is no way to connect two USB-MIDI controllers, sorry. This limitation is in the firmware. For USB host on Arduino, you need a Arduino Due or a USB host shield. And find or write support for USB-MIDI... Kenton MIDI USB Host is an off the shelf solution to convert USB-MIDI to DIN Midi.
