MattilynMattroe 2018-09-26 19:31:12 UTC #5

If it's the same problem I had you can use the TSG/ctrl/multipot 3 object.

Thanks TSG!

I wanted to switch between parameters without the parameter jumping - the hook object does this, but it also waits until you hit the same value to change again. I wanted it so it didn't jump immediately to the knob's current value when you switched, but did jump immediately as soon as the knob was changed after I'd switched... The multi pot object solved it for me. Hope that makes sense!
