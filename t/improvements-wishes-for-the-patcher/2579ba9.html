Yes you only trigger one note. But if I remember correct you can set which note to trigger. There is a small keyboard, similar to the one in Axooti. You set which key to trigger by marking the key you want to play.

Yes sure it is no so good for drumpatches, cause you use more keys for drums. But then again, for drum patches I dont really need to trigger single sounds, cause most of the time the drums are triggered from a sequencer.

For those interested you can download a demo version of Nord G2 that doesnt need the DSP engine. It runs of your computers CPU. The original demo is limited(not all modules are there and no midi in sync etc), but I know that there is a hacked version of the G2 demo that unlocks most of these features. Hacked version is PC only and I havent tried it personally but I have friends who actually sold there G2 and only use hacked version. I am not going to do that, though

Anyway here is link for Nord G2 demo. It only works on Osx 10.6. and down., because of no Rosetta support in newer OSX. Dunno about Pc version compatibility:
http://www.nordkeyboards.com/downloads/legacy/nord-modular-g2
