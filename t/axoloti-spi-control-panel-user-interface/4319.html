Hey Guys,
I've been developing an SPI hardware user interface, mainly for rapid prototype development for our own stuff, something we can re-use all the time.
Someone mentioned that it might be something that would work well with axolti, I must admit, I hadn't heard of this before and have just been reading up about it, looks really cool.

OnBoard: OLED display, 16 pots, 16 sliders, 32 pushbuttons, 48 LEDS and 4 rotary encoders with internal pushbuttons.

Anyways, just letting you know about it. happy to discuss how it could work with this, and we would also be publishing the libraries and firmware as open source.
