Being able to set subpatches to different channels is such a cool feature. It exponentially increases the ways that things done with the Axoloti. You can use synths drums, and samples all at once, and it's great to be able to use the same knobs for the instrument assigned to each channel.

Here's a patch that I use, with some oscillator-based drums and a polyphonic version of @mlinton's Minitaur patch. It also has an audio in which goes through the Repeater effect. The patch is best used with an external sequencer if available, but one can always be built into it as there's plenty of SDRAM available still.

Synth Drums and Poly-Minataur.axp (24.4 KB)

Is anybody else making patches like this? Would love to see what's out there.
