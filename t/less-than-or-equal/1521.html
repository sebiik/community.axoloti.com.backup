It seems like there's some unicode characters in there that the compiler doesn't like. In the names in the XML file you can only use a-z, 0-9, underscore and space since these end up as identifiers in the C++ source code that gets generated from your .axo files (spaces get converted to underscores) . If I'm not mistaken you have given your object the name "â‰¤" - try e.g. "less than or equal" instead.
