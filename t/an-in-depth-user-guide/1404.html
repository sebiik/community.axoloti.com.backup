janvantomme 2016-03-30 11:15:15 UTC #2

Best way to dive in is to check the examples that come with the patcher software. That's how most of us learned about patching. Most objects also have a help patch that shows how to use the object, and how it works. Click the triangle in the upper left corner of the object to see the code or open the help patch.

The book I wrote might also be handy if you want to learn about patching.
