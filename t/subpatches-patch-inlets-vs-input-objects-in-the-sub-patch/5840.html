jaffasplaffa 2019-07-04 13:43:08 UTC #2

If you have the midi stuff in the polyphonic subpatch and you set it to 7 voices you use 7 of every midi objects that you use.

If you have them outside nd route inside via inlet, it uses less cpu/memory.

But to be honest not sure what would be the best functional wise but cpu/memory wise having it outside is the best solution.
