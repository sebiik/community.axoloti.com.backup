Deckard's Dream inspired patch

Patching
krangu 2019-11-24 12:53:59 UTC #1

Not really a faithful or accurate implementation of the features of Deckard's Dream, but it certainly has some rich, sweeping sounds. The rings reverb filter doesn't hurt either. Holding down the keys will allow the LFO which pumps up the sustain to kick in again, causing some nice ambient sounds. Seems to work fine with up to 5 note poly, 6 notes causes audible glitching.
