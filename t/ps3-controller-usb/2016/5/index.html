janvantomme 2016-07-20 10:30:13 UTC #2

Don't think it will be possible, as gamepads usually need drivers. It is possible though to connect thumb joysticks/buttons directly to the Axoloti Core if you solder them to the GPIO pads.

Another option is to connect the gamepad to your computer, and use software to generate MIDI CC and send these to the Axoloti.

johannes 2016-07-20 15:38:26 UTC #4

I think those are generic USB HID game controller devices, no need for a special driver. But HID game controllers are currently not supported in Axoloti, since those are not well supported by the STMicro USB host stack last time I looked, but perhaps someone made an open source implementation already that could be integrated? Should support joysticks, gamepads, as well as slightly more uncommon controllers like the 3dconnexion spacenavigator etc...
