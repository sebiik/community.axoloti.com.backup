johannes 2016-05-25 13:55:33 UTC #6

Ah the patch runs fine while connected, but you want a patch to run standalone?
Just "going live" with a patch only loads it to RAM.
To run a patch standalone, you need to upload it "Upload to sdcard as startup", or "upload to internal flash" (if you do not want to use an sdcard) in the patch menu.
