Hugo 2021-11-29 13:51:57 UTC #2

On arch linux java is controlled by the command

archlinux-java

You can use for example:

archlinux-java status
archlinux-java set java-8-openjdk/jre

Check the arch wiki for details:
https://wiki.archlinux.org/title/Java
