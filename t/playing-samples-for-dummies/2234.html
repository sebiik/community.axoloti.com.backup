thetechnobear 2017-09-22 15:46:27 UTC #6

not really, works here fine.

this error means the directory where the files is supposed to be, doesn't exist (on the sdcard), it doesnt care about the format,
just that it can load the file.

note: FR_NO_PATH means the directory cannot be found , FR_NO_FILE is returned if the filename is not correct

so make sure you have /samples in the root directory of the sdcard, and there isn't something odd about the directory name (e.g. containing spaces etc)
