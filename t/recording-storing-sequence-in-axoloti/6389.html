jaffasplaffa 2020-02-25 14:07:26 UTC #7
plagasul:

The midi player object requires a continuous "on" yellow signal. A manually permanently pressed button object works, but there must be an object that does this automatically, which is it? what object does provide that? It is difficult to find objects by functionality.

You can use a logic/change object for that. If you are using a dial to enter the value, then connect the dial to the blue inlet and then for the yellow inlet, connect the dial to a logic&change object and then to the yellow inlet. Everytime you turn the dial, it sends a trigger to the yellow inlet.
