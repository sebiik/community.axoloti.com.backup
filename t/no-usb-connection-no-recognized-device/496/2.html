thetechnobear 2015-09-24 19:55:28 UTC #2

k, dont panic ....
updating the firmware really cant 'brick' the unit, and the cases of boards failures has been extremely low
(the firmware is stored in a section of memory which can be updated again easily)

are there any LEDs lit on the board? are there any devices shown in windows related to the Axoloti board?

first solution attempt is:

  • boot into DFU mode, this is described in the installation guide as 'rescue mode'
    http://community.axoloti.com/t/installing-on-windows/36
  • do firmware update , using rescue mode

(when you do the above, you can check to see when you put the board into DFU mode, can you see the axoloti board, again as described in the guide)
