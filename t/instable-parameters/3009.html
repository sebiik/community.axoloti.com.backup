thetechnobear 2017-03-05 15:05:20 UTC #4

Ableton clock does jitter when it's externally clocked.

I get the same if I click it from my virus or spectralis.

Basically the issue is Ableton attempts to calculate the bpm and variations caused by midi transports - particularly USB which technically does not have guaranteed timing on most operating systems except when using isosyncronous packets, which is not use for class compliant midi.
