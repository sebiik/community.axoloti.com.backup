Sine Folder V4

Status

Ready to use standalone

Description

Based on the wavefolding implementation topic

  • Gross Folding - Unprecise Folding Amount
  • Fine Folding - More precise Folding Amout
  • Feedback - Feedback the output of the Sine function
  • Wave Input - Fm the initial Sine with the output
  • Bias - Try to correct non symetrical output
  • Amp

Start with Feedback / Wave / Bias and Fine Fold to 0 and Gross Fold to 1.5 for a Classic Sine wave as an Init Patch.

Do not touch the attenuator unless you want to trim the synth.
