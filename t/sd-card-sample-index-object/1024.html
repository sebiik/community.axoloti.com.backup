Yes, improvements are on the way:
* long filename support
* showing directories, hiding hidden files in the file manager
* auto-uploading dependent files to sdcard

Drop-down menu with contents of the sdcard is not planned.
I think local files (not on the sdcard) are "authoritative", uploaded to sdcard when required. That helps when sharing patches.
