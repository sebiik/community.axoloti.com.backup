Hey Kryshe,

I've had huge phase issues with a stereo looper patch and I found out that execution order can be a big problem with stereo processing in axoloti. It could be that your left channel is processed one cycle earlier then the right and thus creating these cancellation problems.

With my patch in the end I moved the left and right channel loopplayers to the same spot, so that they would be processed in the same moment in execution order. so try to move both math/+ to the same place?
