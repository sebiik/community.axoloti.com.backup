johannes 2015-09-20 19:46:03 UTC #2

1: "midi/in/pgm" is for receiving midi program changes. It just receives it, what a patch does with it is up to the patch. "midi/out/pgm" is missing, will add that in the next release.

2: "timer/pulselength", or use "sel/sel b 16" rather than "sel b 16 pulse"

update: midi/out/pgm is in git now for inclusion into the next release.
