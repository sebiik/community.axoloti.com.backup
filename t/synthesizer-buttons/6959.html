janvantomme 2020-11-25 21:01:01 UTC #4

Don't have an axoloti near me, so I haven't tested this patch. Think this might work:

  • gpio/in/digital handes the button press.
  • ctrl/dial b is used to set the note value.
  • midi/intern/note combines the button press and note value into a MIDI message and sends it to the midi/in/keyb object.

Screen Shot 2020-11-25 at 22.04.05.png1268x736 53.9 KB

If you want to play multiple notes at the same time, you should create a polyphonic subpatch from the part of your patch that generates the sound.
