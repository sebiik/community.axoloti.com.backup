Testing Status:
so I upgrade my MBP, here's where I'm at so far, testing is NOT complete, so upgrade at your own risk

Tested with: MBP / 10.12.0 / Axoloti 1.0.11
Overall: No issues encountered so far
Tested: (generally quite limited)
- Axoloti Firmware upload - ok
- Axoloti UI - compile, upload, run patch, adjust parameters - ok
- Axoloti Patch Editing - move objects, delete, move wires, add objects - ok
- Axoloti Libraries - sync , see objects - ok
- Axoloti Object editor - seems ok, but not done any read editing yet.
- CPU - appears to be same as 10.11, as far as i can tell.

so overall, with limited testing, Sierra does not appear to have introduced any new issues. of course more testing is required... and also for longer 'sessions'
