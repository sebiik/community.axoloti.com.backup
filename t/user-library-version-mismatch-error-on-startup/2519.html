johannes 2016-11-12 16:07:33 UTC #2

Here's a solution:

Linux install issue - certificates missing

ok, Ive found the issue! for some reason (yet to work out what) the certificates are no longer included in the java runtime, you can fix this by typing in the following: sudo cp /etc/ssl/certs/java/cacerts /opt/Axoloti/runtime/lib/security/cacerts next step is to work out why Oracle have changed this, note: there is a cacerts.orginal in the same security folder, which is very suspicious
