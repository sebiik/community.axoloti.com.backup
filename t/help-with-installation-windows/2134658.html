Glad to hear!
for Windows 7 winusbcompat seems to be the only way to get our little salamander working. For me, right click on "Axoloti Bulk Interface", selecting "Update Driver Software" and browsing for "winusbcompat.inf" actually turned the "Axoloti Bulk Interface (Interface 2)" into a "generic USB device" - this seems to be one of the inconsistencies Windows 7 brings along. Try and install MIDI Yoke on it - and it'll have you revert back to the good ole "*.inf"-install method as well (introduced with Windows 95, btw.). I'm setting up a notebook (Windows XP... ) atm - and everything works like a charm.
