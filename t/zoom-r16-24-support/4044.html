ZOOM R16/24:
Power on
Press USB Button
Select Audio I/F
Press ENTER
Select EXECUTE
Press ENTER

Use Midi in/out objects

Ch1r0n 2017-12-21 18:19:44 UTC #5

Hi @Gavin,

there is a 'hidden feature' to synchronize Axoloti and R24 in normal operation mode:
Record a couple of minutes from Axoloti ltc/gen object and link this wav file to the click-track on R24. Then you can use the ltc/decode object to sync your patch.
