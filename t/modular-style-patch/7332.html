Looks like you've posted your Axoloti for sale, but if this is still useful to you I suggest you put aside a few hours and go through the tutorial section in the Axoloti menu. You can get there with File > Library > Factory > then Demo or Tutorials. The Tutorials are great because they'll teach you how the parts work on synths in general. It shows some real helpful stuff that extends to Eurorack or basically any type of synth.
