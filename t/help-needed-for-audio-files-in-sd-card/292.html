Hey

I tried last night and these steps worked for me:

  1. Load file in audio editor (did it with Sound forge on Mac)
  2. Save the audiofile to to 16bit 48khz pcm and name it: AMENBR.WAV
  3. Copy the file to the root of the SD Card(What is the name of your SD-Card? Mine is AXOLOTI)
  4. Open Amon is broke patch
  5. Change filename where the wavfile is loaded to: AMENBR.WAV
  6. Set patch for "live".
  7. THIS ONE IMPORTANT. First I had no sound, but his step got it working for me: In upper left corner of the patch, set the variation to anything else than "mute" and you should have sound now.

I checked the BPM of the amenbreak and the tempo is around 137-138 if you want to match it in some way with the LFO tempo control.

Actually all files i loaded worked, when i followed these steps, especially step 7 is important. Also files not converted, worked when I did step 7.

Good luck
