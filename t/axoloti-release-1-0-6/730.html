johannes 2015-12-14 18:50:57 UTC #1

Some highlights:

  • SDCard reader mode! It's quite slow, but avoids the need to eject the microsd-card and transfer it to a cardreader, modify some files, and put it back into Axoloti Core. Entering card reader mode will make the sdcard volume show up just like a card reader does, so you can manipulate files with explorer/finder/... This suspends the Axoloti Patcher communication with Axoloti Core. But you can exit card reader mode by unmounting (eject in windows) the SDCard volume, no need to touch the SDCard or plug any cable to go back to connection with Axoloti Patcher.

  • File associations. You can now double-click an axoloti patch file, this will launch the Axoloti Patcher and open your patch.

  • Text/code editor with undo and syntax highlighting

  • Improved code editor (can save modified objects now, can't create new objects yet)

Download at https://github.com/axoloti/axoloti/releases/latest
