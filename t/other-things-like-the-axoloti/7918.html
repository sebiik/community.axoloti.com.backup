tele_player 2023-02-11 00:16:36 UTC #2

There’s nothing quite like it, but there are interesting things.
If you really like making little synth gadgets, and know one or more of several programming languages, like c/c++,python, PureData, the electro-smith Daisy system is good.
If you like graphically creating new things which run on iPad or Mac, Audulus 4 is amazing. Far more sophisticated graphical patching then Axoloti.

The Electro-Smith Daisy is a similar project and in the same price range.

https://www.electro-smith.com/daisy

It has an active forum: https://forum.electro-smith.com/,
a C++ library: https://electro-smith.github.io/libDaisy/index.html,
a DSP library: https://electro-smith.github.io/DaisySP/index.html

It can also be programmed PureData and even Max.
