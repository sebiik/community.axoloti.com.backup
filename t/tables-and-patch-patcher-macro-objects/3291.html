jaffasplaffa 2017-05-07 21:22:09 UTC #2

if the table located in the main patch is called "drums" ... You should put this in the object reference in the subpatch:

../drums

../ makes the reference to the parent patch... If you are 2 subpatches deep:

../../drums

And so on.....
