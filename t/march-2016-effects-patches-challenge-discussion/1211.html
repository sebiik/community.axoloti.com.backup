Challenge: effect patches - sound modifying patches

create a patch which can change an external (or internal) sound input, and can be influenced/controlled by the user with 5 or less controls, at top of patch, and clearly labelled

Submission window: Tuesday 29th March 00:00 - Thursday 31st March 23:59 (CET)

Voting window: Friday 1st April - Tuesday 5th April 23:59 (CET)

Constraints:

maximum 5 parameters for real time control.
single patch file (axp)

Voting Criteria:

Usefulness
Originality / Extravagance
Patch clarity (is it still possible to see how the patch works?)

Hints/Suggestions:

use patch/patcher rather than axs (due to single file constraint)
the constraint is on main control, other controls can be included for fin ajustements etc

OTO BISCUIT - rebuild in axoloti

rbrt 2016-03-31 18:38:17 UTC #12

here's a last-minute contribution,based on this looper-object I did recently
you need axoloti patcher v 1.08 for it to work

LOOPER-STOMPER

looper-stomp.PNG1406x731 72.1 KB

it does what it does with 4 push-buttons and 1 dial,
RECORD -records a new loop
PLAY/OVERDUB -toggles playback and overdub
SAVE -writes th loop to sd-card
LOAD -loads the loop from sd-card
when you load a loop you've saved,LOOPER-STOMPER does remember the loop's length!
FEEDBACK -when set to anything than 64,the loop will decay gradually
