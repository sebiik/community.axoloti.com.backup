janvantomme 2015-08-16 14:35:19 UTC #2

Take a look at the multiplexer and demultiplexer objects. You can find them in the mux and demux folders in the object selector. A toggle control and counter object can be used to switch between inputs or outputs.

You can find a patch that shows how to use the mux object in the File > Library > tutorials menu: 06_envelopes.axp
