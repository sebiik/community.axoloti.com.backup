jfcharles 2017-09-26 14:49:41 UTC #3

OK, I restarted the Axoloti software, and it works. It seems to be a bug happening in this situation:
- disconnect the USB cable without first disconnecting in the axoloti Window
- then, after you reconnect, the buttons in the File Manager are inactive
Lesson learned: disconnect with a clean toggling off the "Connect" toggle in the axoloti Window.
Also, thanks for the card reader mode tip, I wasn't aware of it!
