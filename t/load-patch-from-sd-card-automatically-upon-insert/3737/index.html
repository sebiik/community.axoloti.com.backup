Load patch from sd card automatically upon insert?

Patching
MC_DETH 2017-09-15 13:01:17 UTC #1

Say I want to use axo core as a standalone audio fx device, with no external controls, powered by an adapter, and I have various sd cards, each one with a different patch on it. How would I make axo load a patch from the sd card automatically, so I would be able to switch patches simply by inserting a different sd card?

strum 2017-09-15 13:44:49 UTC #2

from the patcher upload the patch to the sdcard as the startup patch on each of the sdcards. Insert required card and power cycle the Axo.

A better way would be add a rotary encoder and, optionally, a display so you could select patches from one card. It would be more convenient and faster.

lokki 2017-09-15 15:10:55 UTC #3

why? put them all on one scared and select via program change midi for example. or go with a rotary encoder and display as @strum suggested. but with a midi-controller you can change patches quite easily in standalone mode.
